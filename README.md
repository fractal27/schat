# sChat

Simplest chat webapp that looks good that you could make.

- 1 .go file   ( 170 LOC )
- 1 .css file  ( 138 LOC )
- 1 .js file   ( 86 LOC )
- 1 .sql file  ( 9 LOC )
- 1 .html file ( 34 LOC )

demo: [chat.fdeb.xyz](chat.fdeb.xyz)
